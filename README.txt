
PhrasesPd

PhrasesPd is a MIDI sequencer, controllable by any grid controller capable of using the Monome OSC communications format.

It has an integrated sequence editor, which is controlled by a combination of computer keyboard, MIDI controller, and grid controller.

Installation

1. Put all .pd_lua files into your /pd/extra folder.
2. Make sure all of the program's .pd files are in the same folder as one another. The folder itself can be wherever.
3. Run "phrasespd.pd" in PureData.
